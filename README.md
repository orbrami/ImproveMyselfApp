# ImproveMyself 🚀
**Made By Or Brami**

ImproveMyself is a comprehensive Android lifestyle management application designed to help users track habits, manage workouts, and achieve personal milestones. Built with a focus on consistency and self-improvement, it centralizes all your progress in one place.

---

## ✨ Key Features

* **🎯 Goal Countdown:** Set a target date and track the remaining time with a live countdown timer (Days, Hours, Minutes, Seconds).
* **🏋️ Workouts Manager:** Log exercises with specific Sets & Reps. Data is persisted so your routines are always ready.
* **✅ Habit Tracker:** Create custom daily habits. Check them off in the tracker to keep your streaks alive.
* **📸 Daily Progress Tracker:** 
    * Log healthy eating and workout completion.
    * Add **4 Progress Photos** per day to visually track your transformation.
    * Automatic persistence ensures your daily logs aren't lost on restart.
* **🎬 Movie Goals:** Set a weekly movie target. The app tracks how many movies you've watched this week and shows your remaining goal (e.g., 2/4 movies).
* **🍎 Diet List:** Maintain a persistent list of allowed or recommended foods for your specific nutrition plan.
* **📜 History & Analytics:** 
    * View all past daily records in a clean vertical timeline.
    * **Full-Screen Image Viewer:** Click on any progress photo thumbnail to view it in full size.
    * Individual record deletion for accidental logs.
* **💾 Permanent Storage:** All data is saved to the device's internal storage (SharedPreferences + Gson), ensuring your data survives app closures and phone restarts.

---

## 🛠 Tech Stack

* **Language:** Kotlin 2.0 (K2 Compiler)
* **UI Framework:** Jetpack Compose (Material 3)
* **Navigation:** Jetpack Compose Navigation (Scrollable Bottom Bar)
* **Data Handling:** Gson for JSON serialization/deserialization
* **Image Loading:** Coil (with Persistent URI support for local images)
* **Architecture:** State Hoisting & MutableStateLists for real-time UI updates

## 🚀 Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com
    ```
2. **Open in Android Studio:** Use Android Studio Koala or newer.
3. **JDK Version:** Ensure your project is set to use **JDK 17**.
4. **Run:** Deploy to a physical device or emulator running API 24 or higher.
